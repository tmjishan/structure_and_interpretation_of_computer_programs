Sets of weighted elements In our representation of trees, each non-leaf node contains a set of sym- bols, which we have represented as a simple list. However, the tree- generating algorithm discussed above requires that we also work with sets of leaves and trees, successively merging the two smallest items. Since we will be required to repeatedly ﬁnd the smallest item in a set, it is convenient to use an ordered representation for this kind of set.
