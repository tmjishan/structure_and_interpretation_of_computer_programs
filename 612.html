Notice that looking up a variable always ‘succeeds.’ If lookup-variable- value fails to ﬁnd the variable, it signals an error, as usual. Such a “fail- ure” indicates a program bug—a reference to an unbound variable; it is not an indication that we should try another nondeterministic choice instead of the one that is currently being tried.
