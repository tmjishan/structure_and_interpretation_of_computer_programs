Until now, all our procedures could be viewed as speciﬁcations for comput- ing mathematical functions. A call to a procedure computed the value of the function applied to the given arguments, and two calls to the same procedure with the same arguments always produced the same result. 1 To implement withdraw, we can use a variable balance to indicate the balance of money in the account and deﬁne withdraw as a procedure that accesses balance. e withdraw procedure checks to see if balance is at least as large as the requested amount. If so, withdraw decrements balance by amount and returns the new value of balance. Otherwise, withdraw returns the Insuﬃcient funds message.
