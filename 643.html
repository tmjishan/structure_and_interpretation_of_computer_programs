In this section we give an overview that explains the general structure of the system independent of low- level implementation details. Aer describing the implementation of the interpreter, we will be in a position to understand some of its limitations and some of the subtle ways in which the query language’s logical op- erations diﬀer from the operations of mathematical logic. One way to do this would be to implement the query system as a nondeterministic program, using the amb evaluator of Section 4.3 (see Exercise 4.78).
