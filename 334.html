Each call to rand computes rand-update of the current value of x, returns this as the random number, and also stores this as the new value of x. (define rand (let ((x random-init)) (lambda () (set! x (rand-update x)) x))) Of course, we could generate the same sequence of random numbers without using assignment by simply calling rand-update directly. How- ever, this would mean that any part of our program that used random numbers would have to explicitly remember the current value of x to be passed as an argument to rand-update.
