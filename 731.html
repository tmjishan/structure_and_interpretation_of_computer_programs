All other instruc- tions simply advance pc, making it point to the next instruction in the sequence. Observe that each call to execute calls execute again, but this does not produce an inﬁnite loop because running the instruction execution procedure changes the contents of pc. make-new-machine returns a dispatch procedure that implements message-passing access to the internal state. Notice that starting the machine is accomplished by seing pc to the beginning of the instruc- tion sequence and calling execute.
