(define (analyze-sequence exps) (define (sequentially proc1 proc2) (lambda (env)(proc1 env)(proc2 env))) (define (loop first-proc rest-procs) (if (null? rest-procs) first-proc (loop (sequentially first-proc (car rest-procs)) (cdr rest-procs)))) (let ((procs (map analyze exps))) (if (null? procs)(error "Empty sequence: ANALYZE")) (loop (car procs)(cdr procs)))) To analyze an application, we analyze the operator and operands and construct an execution procedure that calls the operator execution pro- cedure (to obtain the actual procedure to be applied) and the operand execution procedures (to obtain the actual arguments).
